# claude-code-guide-practice

Claude Code 실습 프로젝트입니다.

## 함수 목록

### `add(a, b)`

```python
def add(a: int | float, b: int | float) -> int | float
```

두 숫자를 더합니다.

| 파라미터 | 타입 | 설명 |
|---------|------|------|
| `a` | `int \| float` | 첫 번째 피연산자 |
| `b` | `int \| float` | 두 번째 피연산자 |

**반환값:** `int | float` — 두 값의 합

**사용 예시:**

```python
from ex_pipeline import add

add(1, 2)       # 3
add(1.5, 2.5)   # 4.0
add(-1, 1)      # 0
```

---

### `divide(a, b)`

```python
def divide(a: int | float, b: int | float) -> float
```

두 숫자를 나눕니다.

| 파라미터 | 타입 | 설명 |
|---------|------|------|
| `a` | `int \| float` | 피제수 |
| `b` | `int \| float` | 제수 (0 불가) |

**반환값:** `float` — 나눗셈 결과

**예외:**

| 예외 | 발생 조건 |
|------|----------|
| `TypeError` | `a` 또는 `b`가 숫자 타입이 아닐 때 |
| `ValueError` | `b`가 `0`일 때 |

**사용 예시:**

```python
from ex_pipeline import divide

divide(10, 2)    # 5.0
divide(7, 2)     # 3.5
divide(-6, 3)    # -2.0

# 예외 처리
divide(1, 0)     # ValueError: Cannot divide by zero (b=0)
divide("a", 1)   # TypeError: divide() requires numeric arguments, got str and int
```
